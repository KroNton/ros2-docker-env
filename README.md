# ros2-docker-env
This repository provides script files for running ROS 2 environments in Docker containers. Each script is tailored to a specific ROS 2 distribution.
```
ros2-docker-scripts/
├── README.md
├── LICENSE
├── scripts/
│   ├── run-galactic.sh
│   ├── run-humble.sh
│   ├── run-rolling.sh
│   └── common.sh
├── docker/
│   ├── Dockerfile-galactic
│   ├── Dockerfile-humble
│   └── Dockerfile-rolling
└── .gitignore
```
